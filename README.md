# videotest

当前仓库用于开发“类 PICO VR 互联”的原生原型。

## 当前技术路线

- 前期：**Unity 后置**
- 先做：**Android Native pose sender + Windows Native pose receiver**
- 后做：视频编码/解码链路
- 最后：接入 Unity 作为内容源

## 目录说明

- `architecture_direction.md`：总体架构方向说明
- `phase1_native_plan.md`：第一阶段原生开发计划
- `pico4_ultra_device_bringup.md`：PICO 4 Ultra 真机到手后的联调说明
- `shared-protocol/`：共享协议定义
- `windows-native/`：Windows 原生模块
- `android-native/`：Android Native 模块
- `tools/`：辅助脚本与工具

## 当前状态

已开始搭建第一阶段代码骨架。
